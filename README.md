# 🧩 Maze Solver using A* Search

This project implements the A* (A-star) search algorithm to find the shortest path in a maze.

## 🚀 Features
- A* pathfinding algorithm
- Efficient shortest path calculation
- Easy to understand implementation

## 🧠 Concepts Used
- A* Search Algorithm
- Heuristic Function (f(n) = g(n) + h(n))
- Graph traversal

## ▶️ How to Run
```bash
python maze_solver.py
